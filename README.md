# Calculadora Laboral SV

Sitio estático para estimar aguinaldo, indemnización por despido injustificado, prestación por renuncia voluntaria y salario quincenal en El Salvador.

## Uso local

Abre `index.html` directamente o sirve la carpeta con cualquier servidor HTTP estático.

## Publicación

Puede publicarse sin proceso de compilación en GitHub Pages, Netlify, Cloudflare Pages o cualquier hosting estático. Antes de publicar, agrega el dominio definitivo como `canonical` en `index.html` y crea un `sitemap.xml` con esa URL.

## Alcance

Los resultados son orientativos. Los topes, porcentajes y parámetros legales aparecen como campos editables para poder actualizarlos cuando cambie la normativa.
